# CelloS

Hierarchical AI orchestration framework.

See the `docs/` directory for project documentation.

> **New contributors:** Start with [HANDOFF.md](./HANDOFF.md) for the current state, architecture, and next priorities.
> **Dev guide:** See [docs/acp-guide.md](./docs/acp-guide.md) for how CelloS communicates with worker agents via ACP.

## Quick Start

```bash
pip install .
cellos init
cellos run
```
